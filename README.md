# vue-video-browser

Launch the project with your Youtube API Key :

```
process.env.VUE_APP_YOUTUBE_API_KEY
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
